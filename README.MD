# PyRub
A Rubik's Cube Program built in Python 3.

# How to run this game
Using the terminal of your choosing, run ```python pyrub```

# Dependencies
* Python3 (must also be added to your path)
* ansiwrap==0.8.4
* console-menu==0.8.0
* six==1.16.0
* textwrap3==0.9.2
